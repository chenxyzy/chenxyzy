Windows 一些问题处理

Windows 图标异常

定位到以下路径：HKEY_CURRENT_USER\Software\Classes\Local Settings\Software\Microsoft\Windows\CurrentVersion\TrayNotify
删除PastIconsStream \解决通知栏图标显示异常
删除IconStreams
注意：重启Windows资源管理器生效

Windows 开启始终在任务栏上显示所有图标和通知

定位到HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer
修改EnableAutoTray DWORD值为0，表示开启任务栏显示所有图标和通知；1表示关闭
注意：重启Windows资源管理器生效

Windows 查看注册表对应的本地文件位置

定位到HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\hivelist

Windows7 注册表修改关闭防火墙

定位到HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\SharedAccess\Parameters\FirewallPolicy\PublicProfile
设置EnableFirewall的值为0
定位到HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\SharedAccess\Parameters\FirewallPolicy\StandardProfile
设置EnableFirewall的值为0
注意：重启生效

Windows 网卡驱动后面带#数字

解决驱动名后面带#数字：
定位到HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Network{4D36E972-E325-11CE-BFC1-08002BE10318}\Descriptions
删除Realtek PCIe GbE Family Controller
解决网络适配器连接信息里的本地连接后面带数字：
定位到HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\Network{4D36E972-E325-11CE-BFC1-08002BE10318}
在下面的很多项里展开后查看Connection的Name，找到关键信息后重命名为想要的名称（实时生效），当然也可以删除其他不想要的本地连接信息

Windows10跳过创建用户直接用Administrator身份登录

1、部署完系统进入到区域设置后，按Shift + F10 开启命令行；
2、在命令行 输入 compmgmt，定位到“本地用户和组”并展开；选中用户定位到Administrator，将禁用属性去掉；
3、Ctrl + Shift + F3 进入审计模式（会自动重启）；
4、启动后关闭“系统准备工具XXX” 对话框；
5、将 C:\Windows\System32\svchost.exe 复制到 C:\Windows\System32\oobe\audit.exe（如果提示无权限，修改权限即可；原始audit.exe最好备份下）；
6、重启即可计入系统 shutdown -r -t 0；

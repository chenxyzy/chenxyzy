2、修改默认目录ROOT

path是说明虚拟目录的名字，如果你要只输入ip地址就显示主页，则该键值留为空；
docBase是虚拟目录的路径，它默认的是$tomcat/webapps/ROOT目录，现在我在webapps目录下建了一个myweb目录，让该目录作为我的默认目录。
